## [0.1.0-alpha] - 2026-06-28

### Added

* FastAPI telemetry middleware
* Automatic HTTP request telemetry collection
* Multi-tenant request support
* Request validation
* Runtime metadata support
* Asynchronous in-memory queue
* Background batch processing
* PostgreSQL storage backend
* ClickHouse storage backend
* NoOp storage backend
* Configurable retry mechanism
* Exponential backoff
* Dead Letter Queue (DLQ)
* Runtime metrics collection
* Centralized configuration with `FactConfig`
* Benchmarking utility
* Comprehensive test suite
* GitHub Actions CI pipeline
* Apache License 2.0
* Project documentation
* Python package distribution support
